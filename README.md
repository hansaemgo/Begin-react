## React 연습 

### 01. 작업환경 설정 
    1.	Node.js / Yarn / Editor 설치
    2.	터미널 ( node --version , yarn --version ) 확인
    3.	폴더 경로 찾아가서 ( yarn global add create-react-app )
    4.	npx create-react-app (폴더이름 ) 현재는 begin-react
    5. 열어서 src/hello.js 만들기
    6. imr 눌러서 React import;
